# EDA Project

## Overview
This project focuses on exploratory data analysis (EDA) to understand the dataset, identify missing values, handle data quality issues, detect outliers, perform binning, and analyze relationships between features and the target variable.

## Objectives
- Understand dataset structure
- Check missing values and data quality issues
- Handle datatype corrections
- Detect and analyze outliers
- Perform univariate and bivariate analysis
- Study correlation with target variable

## Tech Stack
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook
